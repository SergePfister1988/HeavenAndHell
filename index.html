Lernbare Skills

Pool an Skills, die ein Held lernen kann, inkl. Freischalt-Level und optionaler Stern-Voraussetzung. Aktive Slots werden später im Spiel aus diesem Pool gewählt.

Wähle oben einen Helden aus, um seine lernbaren Skills zu verwalten.

Attribut-Bäume

Universelle Rauten-Bäume (ANG/DEF/SUP), gemeinsam für alle Helden. Pro Ebene ist nur 1 Feld wählbar; die Wahl bestimmt, welche Felder der nächsten Ebene erreichbar sind.

Backup

Alle Inhalte (Einheiten, Skills, Skillbäume, Vorlagen) als Datei sichern oder wiederherstellen.

Exportieren

Lädt eine JSON-Datei mit allen aktuellen Inhalten herunter. Diese Datei kannst du auch teilen, z.B. um sie im Chat mit Claude zu besprechen.

Importieren / Wiederherstellen

Spielt eine zuvor exportierte JSON-Datei zurück. Bestehende Einträge mit gleicher ID werden überschrieben (aktualisiert), alles andere bleibt unangetastet — es wird nichts vorher gelöscht.
